Переворачивание массива
===

```coffee
a = [1..3] # создание массива а(1,2,3)
b = []  # создание пустого массива b
v = a.length-1 # создание цикла вычитания из массива а
while v >= 0  # функция v: пока v больше 0, из массива а вычитать число 1
   b.push a[v] # присваивание массиву b цикла а
   dir--  # вычитание
console.log b # вывод массива b
```
